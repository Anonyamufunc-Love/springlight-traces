# 春日影踪

追寻“春日影”的轨迹，开启你的创作旅程！本项目是一份专为 After Effects 扩展开发打造的 Bolt-CEP 框架教程，即使你并不想开发 After Effects CEP 扩展，只想开发 Photoshop 扩展的话，也可以从本教程中学习到相关知识。立项于 2025 年 5 月 24 日。无论你是“迷途”的初学者，还是渴望“共鸣”的开发者，都将在这里找到前行的方向。

本文章尽可能的简洁明了的带领你入门 bolt-cep 的框架开发，其基础知识不会过多阐述。

此外，官方仓库的 [README](https://github.com/hyperbrew/bolt-cep/blob/master/README.md) 也很详细，如果不讨厌英文的话，可以在学习春日影踪的同时多多参考官方 README 文档，也能了解到 bolt-cep 的更多知识。

作者碎碎念：由于这是我第一次尝试写教程，经验分享为主，笔力可能还有点“迷途”。所以，如果发现任何错误或不妥之处，请务必温柔地指出哦！一起让这份教程变得更好！

## 关于本地预览

本教程基于 [Docusaurus](https://docusaurus.io/zh-CN/docs)，你只需要运行以下命令，就可以启动本地预览服务器：

```bash
npm install --save-dev
npm start
```
