# 准备工作
欢迎踏上 After Effects 插件开发的旅程！春日影踪教程将引导你基于强大的 Bolt-CEP 框架进行创作。Bolt-CEP 框架拥抱现代 Web 开发技术栈，允许你使用部分 ES6 语法来编写 ExtendScript 代码，并集成了 Vite、TypeScript 和 Sass，并提供了对 React 和 Vue 等流行前端框架的良好支持。

在此之前，春日影踪教程将定义两种 CEP 开发类型：

1. 现代 CEP 开发：基于 Bolt-CEP 框架的插件开发，不兼容太老的版本。
2. 标准/传统 CEP 开发：不适用 Bolt-CEP 框架，只使用 Adobe 自带的示例再做开发，兼容所有版本。

为了确保你能够顺利学习并实践，请提前做好以下准备：

#### 开发环境，以及开发工具

1. 一个合适的编辑器（如Visual Studio Code），或者IDE（WebStorm）
2. Node.js 环境，版本要求 16 及以上（建议 20）
:::warning
从 bolt-cep 2.1 版本开始，由于升级了 React 版本到 19，也许会对 After Effects 2022 版本的兼容性有所影响（GitHub已将最低兼容修改为 After Effects 2024）。
:::
3. After Effects 2022 或更高版本

#### 同时，你也需要一定的编码基础

1. 掌握 JavaScript、TypeScript、HTML 和 CSS 的基本语法和概念。特别是 TypeScript，它是 Bolt-CEP 框架的核心语言，贯穿整个开发过程。
2. 了解最基本的编程概念，如 变量、数据类型、函数、条件语句、循环、数组等。这些是任何编程任务的基础。

本指南假设读者已具备一定的 TypeScript 编程基础。我们将从零开始，手把手地引导你了解 After Effects 插件开发的核心流程与 Bolt-CEP 框架的应用。虽然本教程侧重于核心内容的讲解，可能不会涵盖一个完整项目的所有前端细节，但会深入探讨关键知识，助你从“迷途”走向“共鸣”，在创作的道路上找到前行的方向。
