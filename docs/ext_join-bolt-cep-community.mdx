# 加入 `bolt-cep` 社区

与 `bolt-cep` 的开发团队 `hyperbrew` 以及其他用户交流的最佳方式是加入他们的官方 Discord 服务器。这是一个获取帮助、分享想法和了解项目最新动态的绝佳平台。

**点击这里加入服务器：** [https://discord.com/invite/PC3EvvuRbc](https://discord.com/invite/PC3EvvuRbc)

该 Discord 服务器设有多个频道，其中以下两个频道尤为关键：

*   **`#bolt-cep` 频道：**
    *   获取 `bolt-cep` 的官方更新日志。
    *   参与关于 `bolt-cep` 新功能、改进建议等想法的交流与讨论。

*   **`#dev-help` 频道：**
    *   如果你在开发过程中遇到任何问题，都可以在此频道发布求助帖子。
    *   热心的社区成员会积极提供帮助。
    *   通常情况下，`bolt-cep` 的核心开发者也会在一到两天内关注并回复帖子，协助你解决问题。
